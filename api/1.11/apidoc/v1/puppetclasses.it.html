Param Name Description

Required

POST /api/puppetclasses
Create a puppetclass.

Examples

POST /api/puppetclasses
{
  "puppetclass": {
    "name": "test_puppetclass"
  }
}
201
{
  "puppetclass": {
    "id": 1018350796,
    "name": "test_puppetclass",

    "total_hosts": 0,
    "hostgroups_count": 0,
    "global_class_params_count": 0,
    "variable_lookup_keys_count": 0
  }
}
