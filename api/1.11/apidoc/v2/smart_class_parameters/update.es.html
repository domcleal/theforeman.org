Parámetros

Nombre del parámetro Descripción
id
requerido

Validations:

  • Must be an identifier, string from 1 to 128 characters containing only alphanumeric characters, space, underscore(_), hypen(-) with no leading or trailing space.
smart_class_parameter
requerido

Validations:

  • Must be a Hash
smart_class_parameter[override]
opcional

Si los valores de los parámetros de clase se gestionan mediante Foreman

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
smart_class_parameter[description]
opcional

Descripción de la clase inteligente

Validations:

  • Must be String
smart_class_parameter[default_value]
opcional

Valor a utilizar cuando no existen coincidencias

Validations:

  • Must be String
smart_class_parameter[hidden_value]
opcional

When enabled the parameter is hidden in the UI

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
smart_class_parameter[use_puppet_default]
opcional

No envíe este parámetro mediante ENC. Puppet utilizará el valor definido en el manifiesto Puppet para este parámetro

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
smart_class_parameter[path]
opcional

El orden en el cual se resuelven los valores

Validations:

  • Must be String
smart_class_parameter[validator_type]
opcional

Tipos de valores de validación

Validations:

  • Must be one of: regexp, list.
smart_class_parameter[validator_rule]
opcional

Se utiliza para forzar ciertos valores para los parámetros

Validations:

  • Must be String
smart_class_parameter[override_value_order]
opcional

El orden en el cual se resuelven los valores

Validations:

  • Must be String
smart_class_parameter[parameter_type]
opcional

Tipos de valores de variables

Validations:

  • Must be one of: string, boolean, integer, real, array, hash, yaml, json.
smart_class_parameter[required]
opcional

Si verdadero, se lanzará un error si no hay valor por defecto y ninguna coincidencia de valor

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
smart_class_parameter[merge_overrides]
opcional

Combinar todos los valores coincidentes (sólo de tipo array/hash)

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
smart_class_parameter[merge_default]
opcional

Include default value when merging all matching values

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
smart_class_parameter[avoid_duplicates]
opcional

Eliminar valores duplicados (sólo de tipo array)

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
