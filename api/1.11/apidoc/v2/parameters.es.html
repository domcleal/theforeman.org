Parameters

These API calls are related to nested parameters for a host, domain, host group, operating system. If you are looking for global parameters, go to this link.

>>>

GET /api/hosts/:host_id/parameters
Enumera todos los parámetros de un host

GET /api/hostgroups/:hostgroup_id/parameters
Enumera todos los parámetros de un grupo de hosts

GET /api/domains/:domain_id/parameters
Enumera todos los parámetros de un dominio

GET /api/operatingsystems/:operatingsystem_id/parameters
Enumera todos los parámetros de un sistema operativo

GET /api/locations/:location_id/parameters
Enumera todos los parámetros de una ubicación

GET /api/organizations/:organization_id/parameters
Enumera todos los parámetros de una organización

Ejemplos

GET /api/hosts/host168/parameters
200
